# Frontend App (Vite + React)

This is a minimal Vite + React scaffold generated for a front-end project.

Prerequisites:
- Node.js 18+ and npm or Yarn installed

Quick start:

```bash
cd project
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Notes:
- The dev server runs at `http://localhost:5173` by default.
- To use TypeScript or other frameworks, I can re-scaffold on request.
